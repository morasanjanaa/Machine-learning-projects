# 🚗 Ford Car Price Prediction

A Machine Learning project that predicts the selling price of used Ford cars based on vehicle specifications such as model, year, mileage, fuel type, transmission, engine size, and more. Built with **Python, Scikit-learn, and Streamlit**, covering data preprocessing, model training, and deployment.

🚀 **Live Demo:** https://ford-car-prediction.streamlit.app/
---

## 📌 Features

- Predicts the selling price of Ford cars
- Interactive Streamlit web application
- End-to-end Machine Learning workflow
- Clean and beginner-friendly project structure

## ▶️ Run Locally

### Install dependencies

```bash
pip install -r requirements.txt
```

### Train the model

```bash
python train_model.py
```

### Launch the application

```bash
streamlit run app.py
```
